# Writing Plans (`superpowers:writing-plans`)

## Overview
Processes a specification to generate a comprehensive, executable implementation plan.

## Core Principles
- **Executability**: Plans are execution guides, not just docs.
- **Atomicity**: Tasks must be small (2-5 mins).
- **TDD First**: Every task must include a failing test and a pass condition.

## Structure
Save to `docs/superpowers/plans/YYYY-MM-DD-<name>.md`.
Must include: Goal, Files, Steps, Code Example, Verification, and Commit Message for each task.
