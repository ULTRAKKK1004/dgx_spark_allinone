# Executing Plans (`superpowers:executing-plans`)

## Overview
A disciplined workflow for implementing a plan task-by-task, ensuring high quality and verification at each step.

## The Workflow
1. **Selection**: Pick the next available task.
2. **Setup**: Verify prerequisites and environment.
3. **Execution**: Follow the "Plan -> Act -> Validate" cycle.
4. **Verification**: Run the specified verification command.
5. **Commit**: Commit with the suggested message.

## Mandate
Never skip verification. If a task fails, diagnose and adjust before proceeding.
