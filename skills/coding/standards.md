# Coding Standards (`coding:standards`)

## Overview
Foundation for writing maintainable, secure, and efficient code in any language.

## Principles
- **Clarity**: Code should be self-documenting. Use descriptive names.
- **Security**: Never expose secrets. Validate all inputs.
- **Efficiency**: Minimize context usage and avoid redundant tool calls.
- **Consistency**: Follow existing workspace conventions rigorously.

## Verification
- Run linters and type-checkers (`flake8`, `mypy`, `tsc`, `eslint`).
- Ensure 100% coverage for critical paths.
